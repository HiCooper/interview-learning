description: Kadane 算法 — 贪心 + DP，O(n) 时间 O(1) 空间求最大连续子数组和
---

## 问题描述

给定一个整数数组 `nums`，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**示例:**
- 输入: `[-2, 1, -3, 4, -1, 2, 1, -5, 4]`
- 输出: `6`
- 解释: 连续子数组 `[4, -1, 2, 1]` 的和为 6

**进阶**: 如果你已经实现 O(n) 的解法，尝试使用分治法求解。

## 速查卡

- **Kadane 算法**：贪心+DP，一旦 currentSum < 0 就丢弃前面子数组重新开始
- **核心逻辑**：currentSum = max(num, currentSum + num)；maxSum = max(maxSum, currentSum)
- **O(n) 时间 O(1) 空间**，最简洁高效的子数组问题解法
- **关联理论**：贪心 = 局部最优（负和丢弃）→ 全局最优

## 解法：Kadane 算法

### 思路

维护一个当前子数组和 `currentSum`：

1. 遍历数组，将当前元素加到 `currentSum`
2. 用 `currentSum` 更新最大和 `maxSum`
3. 如果 `currentSum` 变为负数，重置为 0（负数不会对后续子数组和产生正向贡献）

核心思想：**贪心** — 一旦当前和为负，就丢弃前面的子数组，重新开始。

### 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

### Java 实现

```java
public class Solution {
    public int maxSubArray(int[] nums) {
        // 初始化最大和为负无穷，因为数组中的数可能都是负数
        int maxSum = Integer.MIN_VALUE;
        // 初始化当前和为 0
        int currentSum = 0;

        for (int num : nums) {
            // 将当前元素加到当前和上
            currentSum = currentSum + num;
            // 如果当前和大于最大和，则更新最大和
            maxSum = Math.max(maxSum, currentSum);
            // 如果当前和变为负数，则重置为 0
            currentSum = Math.max(currentSum, 0);
        }

        return maxSum;
    }
}
```

## 关键点

- **负数处理**: `maxSum` 初始化为 `Integer.MIN_VALUE`，处理全负数数组的情况
- **重置时机**: `currentSum` 变负时重置为 0，因为负数只会拖累后续和
- **算法名称**: Kadane's Algorithm，是动态规划的简化版（`dp[i] = max(num, dp[i-1] + num)`）
- **分治拓展**: 进阶要求的分治法复杂度为 O(n log n)，不如 Kadane 算法，但适用于需要维护区间信息的场景

## 自测

1. **Kadane 算法为什么 currentSum < 0 时重置为 0 而不是保留？**
   <br/>→ 负数只会拖累后续子数组的和。如果 currentSum 为负，加上下一个元素后的和 < 下一个元素本身。所以最优选择是丢弃前面，从下一个元素重新开始。这是贪心的关键：负和子数组不是任何最优解的组成部分。

2. **进阶：如何用分治法解决最大子序和？复杂度如何？**
   <br/>→ 分治：最大子数组要么在左半边、右半边、或跨越中点。跨越中点的：以中点为起点的左侧最大后缀 + 右侧最大前缀。递归 O(n log n)。比 Kadane O(n) 慢，但展示分治思想，且可扩展到二维矩阵最大子矩阵问题。
