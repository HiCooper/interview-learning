title: 最小路径和
description: 网格最短路径（Minimum Path Sum）— 二维 DP 求解，含多路径回溯
---

## 问题描述

给定一个包含非负整数的 `m × n` 网格 `grid`，找出一条从左上角到右下角的路径，使得路径上的数字总和最小。每次只能向下或向右移动一步。

**示例:**
```
输入:
[1, 3, 4]
[3, 4, 2]
[4, 2, 1]

输出: 最短路径值 = 8
路径: (0,0) → (0,1) → (1,1) → (2,1) → (2,2)
```

## 解法：二维动态规划 + 路径回溯

### 思路

1. **DP 状态定义**: `dp[i][j]` 表示从 `(0,0)` 到 `(i,j)` 的最小路径和
2. **前驱记录**: `pre[i][j]` 记录到达 `(i,j)` 的前驱方向（`-1` 无/边界，`0` 左，`1` 上，`2` 左上均可）
3. **状态转移**:
   - 首行: `dp[0][j] = dp[0][j-1] + grid[0][j]`
   - 首列: `dp[i][0] = dp[i-1][0] + grid[i][0]`
   - 其他: `dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]`
4. **路径回溯**: 从右下角出发，根据 `pre` 数组 BFS 回溯所有可能的最短路径

### 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | O(m × n) |
| 空间复杂度 | O(m × n) |

### Java 实现

```java
import java.util.*;

public class ShortestPathFinder {

    public static void main(String[] args) {
        int[][] inputData = {
            {1, 3, 4},
            {3, 4, 2},
            {4, 2, 1}
        };
        int dis = minPathSumAndPath(inputData);
        System.out.println("最短路径值：" + dis);
    }

    /**
     * 计算最小路径和并返回路径
     */
    public static int minPathSumAndPath(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;

        int[][] dp = new int[m][n];
        dp[0][0] = grid[0][0];
        int[][] pre = new int[m][n]; // -1 无, 0 左, 1 上, 2 左上均可
        pre[0][0] = -1;

        // 首行
        for (int j = 1; j < n; j++) {
            dp[0][j] = dp[0][j - 1] + grid[0][j];
            pre[0][j] = 0; // 来自左
        }
        // 首列
        for (int i = 1; i < m; i++) {
            dp[i][0] = dp[i - 1][0] + grid[i][0];
            pre[i][0] = 1; // 来自上
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (dp[i - 1][j] == dp[i][j - 1]) {
                    dp[i][j] = dp[i - 1][j] + grid[i][j];
                    pre[i][j] = 2; // 上或左均可
                } else if (dp[i - 1][j] < dp[i][j - 1]) {
                    dp[i][j] = dp[i - 1][j] + grid[i][j];
                    pre[i][j] = 1; // 来自上
                } else {
                    dp[i][j] = dp[i][j - 1] + grid[i][j];
                    pre[i][j] = 0; // 来自左
                }
            }
        }

        // 回溯所有可能的最短路径
        List<List<int[]>> allPossiblePaths = findAllPossiblePaths(pre);
        System.out.println("共有路径：" + allPossiblePaths.size());
        for (List<int[]> path : allPossiblePaths) {
            System.out.println(JSON.toJSONString(path));
        }

        return dp[m - 1][n - 1];
    }

    public static List<List<int[]>> findAllPossiblePaths(int[][] pre) {
        int m = pre.length;
        int n = pre[0].length;
        Queue<List<Integer>> queue = new LinkedList<>();

        List<Integer> endPath = new ArrayList<>();
        endPath.add(m - 1);
        endPath.add(n - 1);
        queue.offer(endPath);

        List<List<Integer>> allPaths = new ArrayList<>();

        while (!queue.isEmpty()) {
            List<Integer> currentPath = queue.poll();
            int row = currentPath.get(currentPath.size() - 2);
            int col = currentPath.get(currentPath.size() - 1);

            if (row == 0 && col == 0) {
                allPaths.add(new ArrayList<>(currentPath));
                continue;
            }

            if (pre[row][col] == -1 || pre[row][col] == 2) {
                if (col > 0) addNextStep(queue, currentPath, row, col - 1);
                if (row > 0) addNextStep(queue, currentPath, row - 1, col);
            } else if (pre[row][col] == 0) {
                addNextStep(queue, currentPath, row, col - 1);
            } else if (pre[row][col] == 1) {
                addNextStep(queue, currentPath, row - 1, col);
            }
        }

        return allPaths.stream().map(path -> {
            List<int[]> nodes = new ArrayList<>();
            for (int i = 0; i < path.size(); ) {
                nodes.add(new int[]{path.get(i), path.get(i + 1)});
                i += 2;
            }
            return Lists.reverse(nodes);
        }).collect(Collectors.toList());
    }

    private static void addNextStep(Queue<List<Integer>> queue,
            List<Integer> currentPath, int newRow, int newCol) {
        List<Integer> newPath = new ArrayList<>(currentPath);
        newPath.add(newRow);
        newPath.add(newCol);
        queue.offer(newPath);
    }
}
```

## 关键点

- **pre 数组方向编码**: `-1` 无前驱/边界，`0` 来自左，`1` 来自上，`2` 上下均可（存在多条最短路径）
- **BFS 回溯**: 从终点出发，按 pre 方向反推，处理多条最短路径的场景
- **路径反转**: BFS 收集的是从终点到起点的路径，返回前需反转
