## 问题描述

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。

**示例 1:**
- 输入: `s = "babad"`
- 输出: `"bab"`
- 解释: `"aba"` 同样是符合题意的答案

**示例 2:**
- 输入: `s = "cbbd"`
- 输出: `"bb"`

**提示:**
- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成

## 解法：区间动态规划

### 思路

定义 `dp[i][j]` 表示子串 `s[i..j]` 是否为回文串。

**状态转移:**
- 如果 `s[i] == s[j]` 且满足以下任一条件：
  - `j - i <= 2`（子串长度 ≤ 3，如 `"a"`, `"aa"`, `"aba"`）
  - `dp[i+1][j-1] == true`（内部子串是回文）
- 则 `dp[i][j] = true`，更新最长回文子串的起止位置

**遍历顺序**: `i` 从 `n-1` 到 `0`（逆向），`j` 从 `i` 到 `n-1`（正向），确保 `dp[i+1][j-1]` 在 `dp[i][j]` 之前被计算。

### 复杂度分析

| 指标 | 值 |
|------|-----|
| 时间复杂度 | O(n²) |
| 空间复杂度 | O(n²) |

### Java 实现

```java
class Solution {
    public String longestPalindrome(String s) {
        if (s == null || s.length() < 1) {
            return "";
        }

        int n = s.length();
        boolean[][] dp = new boolean[n][n];
        int start = 0;
        int end = 0;

        // 逆向遍历字符串，移动标 i
        for (int i = n - 1; i >= 0; i--) {
            // 对比标 j
            for (int j = i; j < n; j++) {
                if (s.charAt(i) == s.charAt(j)
                        && (j - i <= 2 || dp[i + 1][j - 1])) {
                    dp[i][j] = true;

                    if (j - i > end - start) {
                        start = i;
                        end = j;
                    }
                }
            }
        }

        return s.substring(start, end + 1);
    }
}
```

## 关键点

- **区间 DP 顺序**: `i` 必须逆向遍历（从右到左），`j` 正向遍历（从左到右），保证子区间 `dp[i+1][j-1]` 已计算
- **长度 ≤ 2 的边界**: `j - i <= 2` 直接判定为回文，无需依赖子区间
- **结果记录**: 只需记录 `start` 和 `end`，最后用 `substring` 返回
- **其他解法**: 还有中心扩展法（O(n²), O(1) 空间）和 Manacher 算法（O(n)）
