## 算法分类速览

### 动态规划
- **最长连续子序列** — 一维 DP，`dp[i]` 表示以 `i` 结尾的最长递增子序列长度
- **最小路径和** — 二维 DP，从左上到右下，记录前驱节点回溯路径
- **背包问题** — 二维 DP，`dp[i][w]` 表示前 `i` 个物品放入容量 `w` 的最大价值
- **最大子序和** — Kadane 算法，贪心 + DP 思想
- **最长回文子串** — 区间 DP，`dp[i][j]` 表示 `s[i..j]` 是否为回文

### 双指针
- **接雨水** — 左右指针向中间收缩，维护 `leftMax` / `rightMax`
- **3Sum / 4Sum** — 排序 + 固定前 N-2 个元素 + 双指针查找

### 哈希表
- **2Sum** — HashMap 存储已遍历元素，一次遍历 O(n)
