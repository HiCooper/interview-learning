## 主题列表

### Java 基础
- **并发编程** — 线程池核心参数与执行流程、AQS 模版方法模式、synchronized 锁升级、ReentrantLock 对比
- **集合框架** — HashMap 数据结构（数组+链表+红黑树）、ConcurrentHashMap 线程安全机制
- **排序算法** — 快速排序、直接插入排序、希尔排序
- **JVM** — 类加载与双亲委派模型、内存布局、垃圾回收算法、常用 GC 收集器
- **NIO** — BIO vs NIO 对比、零拷贝原理

### 中间件
- **Dubbo** — RPC 调用全链路、集群容错、SPI 机制
- **Spring Cloud** — 微服务全家桶、Eureka vs Nacos、配置中心
- **Redis** — 持久化（RDB/AOF）、集群方案、淘汰策略与过期键删除
- **MQ** — 消息可靠性保证、幂等性、顺序消息、分布式事务
- **Kafka** — Topic 与分区、消费者组、流处理平台
- **ZooKeeper** — 数据模型、服务架构、会话机制与顺序保证
- **Netty** — 零拷贝技术、ByteBuf 操作

### 架构设计
- **OAuth 2.0** — 四种授权方式详解
- **DDD 领域驱动设计** — 实体、值对象、聚合、限界上下文与六边形架构
- **事件驱动架构** — CQRS、Event Sourcing 与 Outbox 模式
- **缓存架构** — 缓存策略、多级缓存、缓存三大问题（穿透/击穿/雪崩）、缓存一致性
- **高并发系统设计** — 短链系统、分布式 ID 生成、计数系统、限流策略
- **数据库** — 事务隔离级别、锁机制、索引优化
- **设计模式** — 六大原则 + 常见设计模式
- **分布式一致性** — 交互模式、最终一致性保证

### AI 技术
- **LLM 基础** — Transformer 架构、Tokenization、Prompt 工程与模型选型
- **Agent 开发** — Agent 架构模式（ReAct/Plan-Execute/Multi-Agent）、工具调用、记忆系统
- **RAG** — 检索增强生成：分块策略、向量检索、重排序、高级 RAG
- **MCP** — Model Context Protocol：AI 工具集成标准协议
- **Agent 评测** — 评测指标、LLM-as-Judge、评测框架
- **AI Coding** — AI 辅助编程工作流、Prompt 工程、Agentic Coding
- **上下文管理** — AI Agent 上下文窗口管理策略、业界方案对比

### Leetcode 刷题
- **最长连续子序列** — 一维 DP，O(n²) 解法
- **最小路径和** — 二维网格 DP，含路径回溯
- **背包问题** — 完全背包，二维 DP
- **最大子序和** — Kadane 算法，O(n) 时间 O(1) 空间
- **最长回文子串** — 区间 DP，O(n²) 解法
- **接雨水问题** — 双指针经典题，O(n) 时间 O(1) 空间
- **2Sum / 3Sum / 4Sum** — 哈希表 + 双指针，N 数之和通解
