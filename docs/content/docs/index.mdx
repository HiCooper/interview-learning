## 主题列表

### Java 基础
- **线程池** — ThreadPoolExecutor 核心参数、执行流程、淘汰策略
- **AQS** — AbstractQueuedSynchronizer 核心思想与模版方法模式
- **synchronized** — 监视器模式、锁升级（偏向锁/轻量级锁/重量级锁）
- **集合** — HashMap、ConcurrentHashMap、红黑树
- **排序算法** — 快速排序、直接插入排序、希尔排序
- **JVM** — 类加载、内存布局、垃圾回收、常用 GC 收集器
- **NIO** — BIO vs NIO 对比、零拷贝

### 中间件
- **Dubbo** — RPC 调用全链路、集群容错、SPI 机制
- **Spring Cloud** — 微服务全家桶、注册中心、配置中心
- **Redis** — 持久化、集群、淘汰策略
- **MQ** — 消息可靠性、幂等、顺序消息
- **Kafka** — 分区、消费者组、流处理
- **ZooKeeper** — 数据模型、会话机制
- **Netty** — 零拷贝

### 架构设计
- **OAuth 2.0** — 四种授权方式详解
- **设计模式** — 六大原则 + 常见设计模式
- **数据库** — 事务隔离级别、锁机制、索引优化
- **分布式一致性** — 交互模式、最终一致性保证
